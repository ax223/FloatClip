# Security Policy

FloatClip stores clipboard history locally and does not provide network sync.

## Reporting a Vulnerability

Please do not open a public issue for vulnerabilities that expose private clipboard content, local files, or startup behavior. Instead, report the issue privately to the project maintainer.

If no private contact is listed on the repository, open a minimal public issue asking for a security contact without including exploit details.

## Supported Versions

Security fixes are handled on the latest released version.
